This tool examines the relationship between private deed restrictions and Texas SB 840 in El Paso County, Texas.

Using records collected from the El Paso County, TX Recorder of the Deeds, deed restrictions and covenants were compared to areas authorized for multifamily and residential mixed use by state law.

The analysis finds significant conflict between the state legislation, private deed restrictions, and land use allowances.

See the analysis at: https://hoffmanap.github.io/SB840-Dashboard/
